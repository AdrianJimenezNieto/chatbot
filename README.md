# (ES) Instalación del Chatbot

Para instalar y ejecutar el chatbot, sigue estos pasos desde la línea de comandos:

**Instala las dependencias**
`pip install -r requirements.txt`

**Entrena el modelo**
`python training.py`

**Corre la app**
`python app.py`
